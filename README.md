## 🚀 Getting Started

### Prerequisites

Ensure you have installed:

* Flutter SDK
* Dart SDK
* Android Studio or VS Code
* Android Emulator or Physical Device

### Installation

Clone the repository:

```bash
git clone https://github.com/davidcreated/Spin-and-win-Wheel.git
```

Navigate into the project:

```bash
cd Spin-and-win-Wheel
```

Install dependencies:

```bash
flutter pub get
```

Run the project:

```bash
flutter run
```
